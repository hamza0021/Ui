A website or web site is a collection of network-related web resources, such as web pages and multimedia content, generally identified by a common domain name and published on at least one web server. Notable examples are wikipedia.org, google.com, amazon.com.

Websites can be accessed from a public Internet Protocol (IP) network, such as the Internet, or from a private local area network (LAN) through a Uniform Resource Locator (URL) that identifies the site.

The website has many features and can be used in different fashions. The websites may be personal websites, corporate websites, government websites, organization websites, etc. Web sites are usually dedicated to specific topics and goals, ranging from entertainment and social networking to providing news and education. All publicly accessible collective Web sites are the World Wide Web, but private websites such as corporate websites for employees are usually part of an intranet.

Web pages, which are the building blocks of websites, are documents composed in plain text interspersed with instructions for formatting hypertext markup languages ​​(HTML, XHTML). They can incorporate elements from other websites using appropriate markup anchors. Web pages are accessible and transferred using the Hypertext Transfer Protocol (HTTP), which can optionally employ encryption (HTTP Secure, HTTPS) to provide security and privacy for the user. The user application (often a web browser) renders the page content according to the HTML markup instructions on the display terminal.

Hyperlinks between web pages conveys the structure of the site to readers and guide the navigation on the site. Site navigation often begins with a home page that contains a web content directory for the site. Some websites require user registration or a subscription to access the content. Examples of subscription websites include many business sites, news websites, academic websites, gaming websites, file sharing websites, message boards, web e-mails, social media websites, websites providing real-time stock market data, as well as sites providing various other services. End users can access websites on a variety of devices, including desktops and laptops, tablets, smartphones, and smart TVs.
